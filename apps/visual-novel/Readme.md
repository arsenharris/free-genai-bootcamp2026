## Background Image Generation (Spanish scenes)

Use image generation tools to create backgrounds and character art suited to Spanish settings (plazas, cafés, apartamentos). Prompt models for Mediterranean/European interiors, warm daylight, and culturally appropriate props (posters, signage in Spanish).

- Amazon Nova Canvas can produce good backgrounds with careful prompts.
- SDXL may require tailored prompts for European-style interiors.
- Grok 3 produced strong backgrounds in testing; if used manually, consider applying a consistent color/style filter.

## Image Expression

For character posing and subtle expression adjustments consider tools like FacePoke or similar face/pose spaces. These help create natural head turns and simple poses for dialogue sprites.
